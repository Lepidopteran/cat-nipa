# Cat NIPA

Pseudo alternative to [nipa](https://github.com/Wilhansen/nipa) made in Rust.

Currently in the experimental phase.

The primary focus is on extracting files from NPA archives.

Depending on user interest and demand, future updates may include support for creating NPA archives.

## FAQ

<details>
<summary>Why the name?</summary>
I like cats, and when thinking of a name for this project and trying the project <a href="https://github.com/Wilhansen/nipa">nipa</a>, I thought of catnip.
</details>

<details>
<summary>Why make this?</summary>
I couldn't use <a href="https://github.com/Wilhansen/nipa">nipa</a> easily on linux, so I decided to see if I could make it.
</details>
